# NPM-Starter

> Starting point for modules being published to npmjs.com

## Getting Started

### Install

```bash
git clone git@github.com:servexyz/npm-starter
```

### Use

**Test**

> Can run test in isolation

```bash
npm run test
```

**Build**

> Can run build in isolation

```bash
npm start
```
